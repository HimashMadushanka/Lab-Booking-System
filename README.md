# Lab Booking System

A web-based system for managing laboratory bookings. Built with PHP, HTML, CSS and MySQL.  
Users can register/login, view and book available labs, track bookings, and administrators can manage lab resources, bookings and generate reports.

## Features

- User registration and authentication  
- User dashboard: view available labs, make bookings, view/current bookings  
- Administrator panel: manage labs, review bookings, generate PDF reports  
- Feedback mechanism  
- Booking history and status tracking  
- Secure logout / session management

---

## Technology Stack

- **Backend**: PHP  
- **Frontend**: HTML, CSS  
- **Database**: MySQL  
- **Libraries/Tools**: FPDF (for PDF generation)  
- **Environment**: Web server (e.g., Apache) with PHP support
